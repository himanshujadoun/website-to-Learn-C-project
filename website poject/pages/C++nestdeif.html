nested if

A nested if statement in C++ is when you place one if statement inside another. This allows for more complex conditional branching in your code. The inner if statement is executed only if the condition of the outer if statement is true . Here's a short example:

Syntax

if (condition1) {

      // Code to be executd if condition 1 is true

    if (condition2) {

             // Code to be executed if both condition1 and condition2 are true

               }

         else{

              // Code to be executed if condition1 is true and condition2 is not true

               }


else{

// Code to be executed if both condition1 is true

}

}

Example

#include

using namespace std

int main(){

  int num =-1;

  if(num>=0){

            if(num==0){

                          cout<< num<<" is zero";

                      }

             else{

                      cout<< num<<" num is positive";

                      }

         }

   else{

         cout << num <<" is negative";

        }

retun 0;

}

Explain

This C++ code defines a program that checks whether the integer variable 'num' is positive, zero, or negative. It uses nested if statements. If 'num' is greater than or equal to 0, it checks if it's zero or positive and prints the corresponding message. If 'num' is negative, it prints a message indicating that.
