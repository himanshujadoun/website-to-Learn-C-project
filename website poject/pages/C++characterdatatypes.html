C++ characters Data Types

Character Type

The char data type is used to store a single character. The character must be surrounded by single quotes, like 'A' or 'c':

Example

#include

using namespace std

int main(){

char myGrade = 'B'; cout << myGrade;

return 0;

}

NOTE: can use ASCII values to display certain characters:


Example

#include

using namespace std

int main(){

char a = 65, b = 66, c = 67;

cout << a;

cout << b;

cout << c;

return 0;

}
