C++ Break

break in C++ is used to exit a loop or switch statement prematurely, transferring control to the statement following the loop or switch.

The break statement can also be used to jump out of a loop.

Example

#include

using namespace std;

int main() {

for (int i = 0; i < 10; i++) {

if (i == 4) {

break;

}

cout << i << "\n";

}

return 0;

}

C++ Continue

In break, "continue" is a control statement that skips the remaining code in a loop iteration and proceeds to the next iteration.

Example

#include

using namespace std;

int main() {

for (int i = 0; i < 10; i++) {

if (i == 4) {

continue;

}

cout << i << "\n";

}

return 0;

}
