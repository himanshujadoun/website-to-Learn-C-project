while Loops

while loops in C++ are used when you want to repeatedly execute a block of code as long as a specific condition remains true.

Syntax

while(condition){

      //code to be executed untill the condition is true

}

Example

#include

using namespace std;

int main(){

int a=0;

while(a<10){

     cout<< a<< endl;

     a++;

           }

   return 0;

}
