C++ Pointers

A pointer is a special type of variable that stores the memory address of an variable and object.

Syntax

data_datatype_of_pointer* pointer_name = & Normal_variable_that_address_to_be_store

Example

#include

using namespace std;

int main(){

int pt = 38;


// & is used to store address of pt variable

int &p = pt;


//acces the value of pt using pointer p

cout << *p << endl;

return 0;

}
