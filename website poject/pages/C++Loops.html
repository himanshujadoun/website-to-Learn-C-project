C++ Loops

In C++, Loops are control structures used to execute a block of code repeatedly.

Loops are handy because they save time, reduce errors, and they make code more readable.

Types of Loops in C++

  1. for Loop
  2. do Loop
  3. do while Loop

for Loop

When you know exactly how many times you want to loop through a block of code, use the for loop instead of a other loop:

Syntax

for (initialization; condition; update) {

// body of-loop

}

Example

#include

using namespace std;

int main(){

for(int a=1; a<10;a++){

                  cout<<"Welcome DEI "<< endl;

                      }

return 0;

}
