C++ Arrays

Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.

To declare an array, define the variable type, specify the name of the array followed by square brackets and specify the number of elements it should store:

Elements of an array can be accessed arbitrarily using its indices. They can be used to store a collection of any type of primitive data type, including int, float, double, char, etc. An array in C/C++ can also store derived data types like structures, pointers, and other data types, which is an addition. The array representation in a picture is provided below.

Advantage of Array in C++

  • Code Optimization (less code)
  • Random Access
  • Easy to traverse data
  • Easy to manipulate data
  • Easy to sort data etc.

Disadvantage of Array in C++

  • Fixed size
  • Lack of flexibility
  • Wasted Memory

    There are 2 types of arrays in C++ programming:

    1. Single Dimensional Array
    2. Mutlidimensional Array

    C++ Single Dimensional Array

    A Singal Dimensional Array is a group of elements having the same data type which are stored in a linear arrangement under a single variable name.

    Syntax

    data_datatype <"String_name"> [no of data to store ] = { data_1 , data_2 ,data_2 , data_4 };

    Example

    #include

    using namespace std;

    int main()

    {

    int arr[5]={10, 0, 20, 0, 30}; //creating and initializing array

    //traversing array

      cout<< arr[0]<< endl;

      cout<< arr[1]<< endl;

      cout<< arr[2]<< endl;

      cout<< arr[3]<< endl;

      cout<< arr[4]<< endl;

    return 0

    return 0;

    }

    C++ Mutlidimensional Array

    To declare a Multidimensional Array, define the variable type, specify the name of the array followed by square brackets which specify how many elements the main array has, followed by another set of square brackets which indicates how many elements the sub-arrays have:

    Syntax

    data_datatype <"String_name"> [no of rows][no of column]= {

          {<" Data in rows ">},

          { <" data in columns">}

    };

    Example

    #include

    using namespace std;

    int main() {

    string letters[2][4] = {

    { "A", "B", "C", "D" },

    { "E", "F", "G", "H" }

    };

    cout << letters[0][0];

    return 0;

    }

    Note:

    You can update Array at special index of Array:

    Syntax

    array_name [index of value that to be change ] = { value to be change };

    Example

    #include

    using namespace std;

    int main(){

       int dt[5]={21,44,53,35,43,};

         // value at index 4 is change from 43 to 10

       dt[4]=10;

        cout<<"Value of dt array at index 4 is: "<< dt[4]<< endl;

    return 0;

    }
