C++ boolean Data Types


Boolean Type

A boolean data type is declared with the bool keyword and can only take the values false or true.

When the value is returned, true = 1 and false = 0.

Example

#include

using namespace std

int main(){

bool isCodingFun = true;

bool isFishTasty = false;

cout << isCodingFun << "\n";

cout << isFishTasty;

return 0;
}
