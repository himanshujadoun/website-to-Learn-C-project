Mutliple declare variables

To declare more than one variable of the same type , use a comma-separated list:

Syntax

datatype variable_name = value, variable_name1 = value1, variable_name2 = value2, variable_name3 = value3 ;

Example

#include

using namespace std;

int main(){

int x=10,y=20,z=30;

cout<<"value x "<< x<< endl;

cout<<"value y "<< y<< endl;

cout<<"value z "<< z<< endl;

return 0;

}

One Value to Multiple Variables

You can also assign the same value to multiple variables in one line:

Example

#include

using namespace std;

int main(){

int x, y, z;

x = y = z = 50;

cout << x + y + z;

return 0;

}
